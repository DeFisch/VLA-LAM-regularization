# Modified LIBERO RLDS Dataset

Download this dataset from HuggingFace:

```bash
git clone https://huggingface.co/datasets/openvla/modified_libero_rlds
```

Or use the HuggingFace CLI:

```bash
huggingface-cli download openvla/modified_libero_rlds --repo-type dataset --local-dir ./modified_libero_rlds
```
